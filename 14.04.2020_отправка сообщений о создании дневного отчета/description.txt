Отправка уведомлений при создании пользователями дневного отчета (если пользователь находится в группе отслеживаемых) всем сотрудникам из группы руководителей (выбирается в options.php):

1. ОБЯЗАТЕЛЬНОЕ условие : группы должны быть выбраны в options.php, иначе модуль не будет работать!

2. В панели старта рабочего дня есть окно отчета за день. При создании такого отчета сотрудником, который состоит в группе отслеживаемых, отправляется уведомление всем, кто состоит в группе получателей уведомленийю

Результат:
1. В options.php настроен выбор групп пользователей.
2. Установлена привязка к событию OnAfterTMReportAdd.
3. Проверяется:
	- заполнение полей с ID групп;
	- принадлежность создающего отчет группе контролируемых;
	- если группа получателей уведомлений не пуста, то каждый получает уведомление со ссылкой на отчет.
